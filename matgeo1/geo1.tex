\textbf{Question}:\\
The construction of triangle $ABC$, given that $BC$ = 6$cm$, $\angle C$ = 60$^\circ$ is possible when difference of $AB$ and $AC$ is equal to
\begin{enumerate}
    \item 3.2$cm$
    \item 3.1$cm$
    \item 3$cm$
    \item 2.8$cm$
\end{enumerate}

\textbf{Solution: }

\begin{enumerate}
    \item  $BC$  $=$ 6$cm$
 \item $\angle C = 60^\circ$
  \item let $d = \abs{ AB-AC}$
\end{enumerate}
    

    The Law of Cosines is given by:
\[
AB^2 = BC^2 + AC^2 - 2 \cdot BC \cdot AC \cdot \cos(\angle C)
\]
Given $BC = 6  \text{cm}$ and $\angle C = 60^\circ$, we know:
\[
\cos(60^\circ) = 0.5
\]
Then , 
\[
6AC = 36 - AC^2 - AB^2
\]
Let the difference between $AB$ and $AC$ be $d$,

then,
there are two possibilities,
\[
 1) AB = AC + d ;
\]
\[
2) AC = AB + d
\]
1)  let $AB = AC + d$.

Substitute these values into the Law of Cosines equation:
\[
6AC = 36 - AC^2 - (AC + d)^2 
\]
\[
AC = \frac{36 - d^2}{6 + 2d}
\]
for the first case,
For $AC$ to be positive, $d$ $<$ 6$cm$.
all the options satisfy the following condition.
therefore,d can take the values 3.2$cm$,3.1$cm$,3$cm$,2.8$cm$.

2) let $AB = AC - d$
substitute these values in the law of cosines equation:
\[
6AC = 36 - AC^2 - (AC - d)^2 
\]
\[
AC = \frac{36 - d^2}{6 - 2d}
\]
for the second case,\\
For $AC$ to be positive, $d$ $<$ 3$cm$.
only one option satisfies this condition.
therefore,d can only take the value 2.8$cm$.
For each of the $d$ value there are different coordinates of $A$.
